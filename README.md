# ArchaeoInsights: Unveiling Patterns and Trends through Data Science

This repository contains data analysis code for exploring archaeological site data. The dataset used for this analysis includes information about various archaeological sites, including their locations, species found, and other relevant details.

## Dataset

The dataset is stored in a CSV file at `data/nerd.csv`. 

## Analysis

The analysis code is written in Python and includes the following operations:
- Data loading and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization
- Frequency analysis of species and sites

## Instructions

To run the analysis, follow these steps:
1. Clone this repository to your local machine.
2. Install the required Python packages specified in `requirements.txt`.
3. Run the notebook final.ipynb.

## Dependencies

The analysis code requires the following Python packages:
- pandas
- matplotlib
- seaborn
- folium (for mapping)

You can install the dependencies using pip:

```
pip install -r requirements.txt
```
